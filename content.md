**아이콘**: 🗂️

**본문**: 매물 분석·광고문·고객 응대를 따로따로 챗GPT 켜서 하지 마세요. **한 워크스페이스에 묶어 두고** 필요할 때마다 꺼내 쓰는 도구입니다.

**아이콘**: 🧠

## ③ How — 사용 방법 3단계

**섹션 헤딩**: "그래서 어떻게 시작합니까?"
**섹션 서브**: 진짜 3단계. 설치부터 첫 결과물까지 약 10분.

### Step 1 — Claude Desktop 설치 + 유료 플랜 (약 5분)
**제목**: Claude Desktop 설치 + 유료 플랜
**본문**:
1. `claude.com/download` 접속 → **Claude Desktop** (macOS / Windows) 설치
2. 이메일·구글 계정으로 로그인
3. 설정에서 **Pro 플랜 ($20/월) 이상** 결제 — Cowork은 무료 플랜에서는 사용할 수 없음

> 💡 **왜 데스크탑 앱인가**: Cowork은 사무소 컴퓨터의 파일·이메일·다른 앱에 접근해 일을 처리하는 도구라 데스크탑 앱 전용. (macOS + Windows 모두 지원, 2026-02-10부터 윈도우도 동일 기능)

### Step 2 — 워크스페이스(Project) 만들기 (약 2분)
**제목**: "내 부동산 사무소" 작업 공간 만들기
**본문**:
1. Claude Desktop 좌측 사이드바 `Projects` → `+` 클릭
2. "Start from scratch" 선택 → 이름: "부동산 하는 도시아재" (본인 사무소 이름으로)
3. **Instructions** 칸에 5~8줄 입력:
   ```
   나는 서울 ○○구 공인중개사입니다.
   매물 분석·광고·고객 응대 작업을 자주 합니다.
   답변은 카톡으로 그대로 복사할 수 있는 자연스러운 한국어로 해주세요.
   숫자는 정확하지 않으면 "확인 필요"라고 표시해 주세요.
   ```

> 💡 **팁**: Instructions는 5~8줄 이내 짧게가 공식 권장. 이 짧은 한 단락이 "한 번 알려주면 계속 기억"의 핵심.

### Step 3 — 첫 프롬프트 (약 3분)
**제목**: 매물 한 건 분석시켜 보기
**본문**: 방금 만든 Project 안에서 새 작업을 만들고 다음을 그대로 붙여넣어 보세요.
```
서울 강남구 역삼동 ○○아파트 84㎡, 매매 25억 매물입니다.
같은 단지 최근 거래가, 주변 시세, 강점·약점 3개씩,
그리고 카톡으로 보낼 광고 문구 1개 만들어주세요.
```
**기대 결과물**: 단지 시세표 · 강점/약점 리스트 · 카톡용 광고문 한 단락이 한 번에 나옵니다.

## ④ Apply — 공인중개사 업무에 끼우는 4가지 진입점

**섹션 헤딩**: "그래서 내 일 어디에 씁니까?"
**섹션 서브**: 4가지만 기억하세요. 나머지는 응용입니다.

| 진입점 | 한 줄 설명 | 시간 절약 |
|---|---|---|
| 🏢 **매물 분석** | 주소 한 줄 → 시세·등기·강점/약점 리포트 | 30분 → 3분 |
| 📣 **광고문 작성** | 분석 결과 → 블로그·당근·인스타 3채널 광고 카피 | 1시간 → 5분 |
| 💬 **카톡 응대 초안** | 고객 문의 → 분류 + 자연스러운 답변 초안 | 10분 → 30초 |
| 📋 **고객 보고서** | 임장 메모 → 깔끔한 PDF용 보고서 | 1시간 → 10분 |

> 💡 **모두 같은 패턴**: "내 사무소" 워크스페이스에 정보 한 번 넣어두면, 4가지 모두 "한 줄 입력 → 한 화면 출력"입니다.

## ⑤ Cases — 추천 사례 카드 (인터랙티브 + Dispatch 시연)

**섹션 헤딩**: "직접 눌러보세요"
**섹션 서브**: 공인중개사 업무에서 자주 만나는 시나리오 4개입니다.

### 사례 1 (인터랙티브 카드)
**제목**: "주소만 넣고 임장 가방 챙기기"
**상황**: 오전 10시, 고객이 갑자기 "광화문 A아파트 볼 수 있어요?"
**Before**: 네이버부동산 · 호갱노노 · 디스코 3개 창 동시에 열고 30분
**After (카드 펼침)**:
1. Cowork에 주소 한 줄 입력 → 3분 만에 시세·근거리 비교·교통 정리 출력
2. 그대로 카톡으로 고객에게 "오시는 동안 보세요" 전송

### 사례 2 (인터랙티브 카드)
**제목**: "한 번 적은 광고, 3채널로 자동 변환"
**상황**: 같은 매물 광고를 블로그·당근·인스타에 3번 다시 쓰는 중
**Before**: 카피 3번 변형 + 해시태그·길이 조정 = 1시간
**After (카드 펼침)**:
1. 매물 정보 1번 입력 → 블로그(긴 글) · 당근(짧고 친근) · 인스타(이모지) 3개 출력
2. 그대로 복사·붙여넣기

### 사례 3 (인터랙티브 카드)
**제목**: "카톡 문의 폭주 → 분류 + 답변 초안"
**상황**: 광고 하나 올렸더니 카톡 문의 50개
**Before**: 1개씩 읽고 답변 = 2시간 + 놓치는 고객
**After (카드 펼침)**:
1. 문의 50개 붙여넣기 → "진성/단순문의/스팸" 자동 분류
2. 진성 고객 답변 초안까지 같이 → 검토·수정만 하면 끝

### 사례 4 — 🔥 Cowork Dispatch (베타) 시연 — 📱 폰 → 💻 사무소 → 📲 폰
**제목**: "📱 폰에서 보내면 → 💻 사무소 컴퓨터가 처리 → 📲 폰으로 결과"
**섹션 서브**: 외근·임장 중에 폰으로 한 줄 보내면, 사무소 Claude Desktop이 알아서 실행하고 완성된 결과를 다시 폰으로 보내줍니다. (Anthropic Cowork 공식 기능, 2026-03-17 출시 베타)

**3단 흐름 시각화 (가로 3 카드)**:
1. 📱 **1. 폰에서 보내기** — 외근·임장 중에 Claude 모바일 앱으로 작업 한 줄 작성
2. 💻 **2. 데스크탑이 실행** — 사무소 컴퓨터의 Claude Desktop이 파일·이메일·도구에 접근해 자동 처리
3. 📲 **3. 폰으로 결과** — 완성된 결과(분석·광고·응대·보고서)가 폰으로 도착

**🚗 임장 가는 차 안 · 폰 입력 예시**:
```
광화문 한일아파트 84㎡, 매매 18억 매물.
오후 임장 예정, 고객은 30대 신혼부부.
시세 분석·광고 카피·임장 직전 카톡 안내·임장 후 보고서 템플릿
4가지 한 번에 만들어서 준비해 둬.
```

**사무소 데스크탑이 한 번에 만들어내는 4개 산출물** (각 카드 클릭 시 데스크탑이 받은 지시 펼침):
- 💻 **산출물 A (시세 분석)**: 같은 단지 최근 5건 거래·근거리 비교
- 💻 **산출물 B (광고 카피)**: 신혼부부 톤 광고문 (블로그·당근·인스타)
- 💻 **산출물 C (카톡 응대)**: "임장 시 체크포인트" 안내 메시지
- 💻 **산출물 D (고객 보고서)**: 임장 후 PDF용 1장 요약

> ⚠️ **베타 한계**: Dispatch는 현재 research preview. 사무소 컴퓨터가 켜져 있고 Claude Desktop이 실행 중이어야 작동. 절전·앱 닫힘 상태에서는 불가. 파일 데이터 추출·메일 요약·문서 작성은 안정적이고, 일부 서드파티 앱 조작은 아직 불안정.

> 💡 **공식 정의** (Anthropic Cowork): *"Message from your phone. Claude works on your computer. You get the result."* — 폰으로 메시지, 컴퓨터가 작업, 결과를 받음. 임장·외근이 많은 공인중개사 업무에 가장 잘 맞는 사용법.
