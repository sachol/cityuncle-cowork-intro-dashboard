HOW · 시작하기

그래서 어떻게 시작합니까?

진짜 3단계. 설치부터 첫 결과물까지 약 10분.

01

Claude Desktop 설치 + 유료 플랜 (약 5분)

  1. claude.com/download 접속 → Claude Desktop (macOS / Windows) 설치
  2. 이메일·구글 계정으로 로그인
  3. 설정에서 Pro 플랜 ($20/월) 이상 결제 — Cowork은 무료 플랜에서는 사용할 수 없습니다
💡 왜 데스크탑 앱인가: Cowork은 사무소 컴퓨터의 파일·이메일·다른 앱에 접근해 일을 처리하는 도구입니다. 그래서 웹·모바일이 아닌 데스크탑 앱 전용입니다. (macOS + Windows 모두 지원, 2026-02-10부터 윈도우도 동일 기능)
02

"내 부동산 사무소" 작업 공간 만들기 (약 2분)

  1. Claude Desktop 좌측 사이드바 Projects → + 클릭
  2. "Start from scratch" 선택 → 이름: "부동산 하는 도시아재" (본인 사무소 이름으로)
  3. Instructions 칸에 다음 5~8줄을 입력:
나는 서울 ○○구 공인중개사입니다.
매물 분석·광고·고객 응대 작업을 자주 합니다.
답변은 카톡으로 그대로 복사할 수 있는 자연스러운 한국어로 해주세요.
숫자는 정확하지 않으면 "확인 필요"라고 표시해 주세요.
💡 팁: Instructions는 5~8줄 이내 짧게가 공식 권장. 길게 쓸 필요 없습니다. 이 짧은 한 단락이 "한 번 알려주면 계속 기억"의 핵심입니다.
03

매물 한 건 분석시켜 보기 (약 3분)

방금 만든 Project 안에서 새 작업을 만들고, 다음을 그대로 붙여넣어 보세요:

서울 강남구 역삼동 ○○아파트 84㎡, 매매 25억 매물입니다.
같은 단지 최근 거래가, 주변 시세, 강점·약점 3개씩,
그리고 카톡으로 보낼 광고 문구 1개 만들어주세요.
기대 결과물: 단지 시세표 · 강점/약점 리스트 · 카톡용 광고문 한 단락이 한 번에 나옵니다.
APPLY · 적용

그래서 내 일 어디에 씁니까?

4가지만 기억하세요. 나머지는 응용입니다.

🏢

매물 분석

주소 한 줄 → 시세·등기·강점/약점 리포트

30분 → 3분
📣

광고문 작성

분석 결과 → 블로그·당근·인스타 3채널 광고 카피

1시간 → 5분
💬

카톡 응대 초안

고객 문의 → 분류 + 자연스러운 답변 초안

10분 → 30초
📋

고객 보고서

임장 메모 → 깔끔한 PDF용 보고서

1시간 → 10분

💡 모두 같은 패턴: "내 사무소" 워크스페이스에 정보 한 번 넣어두면, 4가지 모두 "한 줄 입력 → 한 화면 출력"입니다.

CASES · 사례

직접 눌러보세요

공인중개사 업무에서 자주 만나는 시나리오 4개입니다.

BEFORE

네이버부동산 · 호갱노노 · 디스코 3개 창 동시에 열고 30분

AFTER
  1. Cowork에 주소 한 줄 입력 → 3분 만에 시세·근거리 비교·교통 정리 출력
  2. 그대로 카톡으로 고객에게 "오시는 동안 보세요" 전송
BEFORE

카피 3번 변형 + 해시태그·길이 조정 = 1시간

AFTER
  1. 매물 정보 1번 입력 → 블로그(긴 글) · 당근(짧고 친근) · 인스타(이모지) 3개 동시 출력
  2. 그대로 복사·붙여넣기
BEFORE

1개씩 읽고 답변 = 2시간 + 놓치는 고객

AFTER
  1. 문의 50개 붙여넣기 → "진성/단순문의/스팸" 자동 분류
  2. 진성 고객 답변 초안까지 같이 → 검토·수정만 하면 끝
🔥 CASE 04 · DISPATCH (베타)

📱 폰에서 보내면 → 💻 사무소 컴퓨터가 처리 → 📲 폰으로 결과

외근·임장 중에 폰으로 한 줄 보내면, 사무소 Claude Desktop이 알아서 실행하고 완성된 결과를 다시 폰으로 보내줍니다. 결과 받으러 사무소에 돌아갈 필요가 없습니다.

📱
1. 폰에서 보내기

외근·임장 중에 Claude 모바일 앱으로 작업 한 줄 작성

💻
2. 데스크탑이 실행

사무소 컴퓨터의 Claude Desktop이 파일·이메일·도구에 접근해 자동 처리

📲
3. 폰으로 결과

완성된 결과(분석·광고·응대·보고서)가 폰으로 도착

🚗 임장 가는 차 안 · 폰 입력 예시
광화문 한일아파트 84㎡, 매매 18억 매물.
오후 임장 예정, 고객은 30대 신혼부부.
시세 분석·광고 카피·임장 직전 카톡 안내·임장 후 보고서 템플릿
4가지 한 번에 만들어서 준비해 둬.

아래 4개 카드는 폰에서 보낸 한 줄로 사무소 데스크탑이 한 번에 만들어내는 산출물입니다.

💻 산출물 A

시세 분석

📊

같은 단지 최근 5건 거래·근거리 비교 분석

다음 매물의 시세 분석을 해줘.
같은 단지 최근 거래, 인근 단지 비교,
강점·약점 각 3개.
- 단지: 광화문 한일아파트
- 면적: 84㎡
- 가격: 매매 18억
💻 산출물 B

광고 카피

📣

신혼부부 톤 광고문 (블로그·당근·인스타)

위 매물을 30대 신혼부부 톤으로 광고해줘.
3개 채널: 블로그(긴 글), 당근(짧고 친근),
인스타(이모지+해시태그).
💻 산출물 C

카톡 응대

💬

"임장 시 체크포인트" 안내 메시지 초안

고객에게 보낼 임장 직전 카톡 메시지를 만들어줘.
신혼부부 톤, 체크포인트 5개 포함,
카톡으로 그대로 보낼 수 있게.
💻 산출물 D

고객 보고서

📋

임장 후 PDF용 1장 요약 보고서 템플릿

임장 직후 고객에게 보낼 1장 요약 보고서를 만들어줘.
형식: 매물 요약·강점/약점·총평·다음 액션.
PDF 출력용으로 깔끔하게.
⚠️ Dispatch는 현재 베타(research preview)입니다. 사무소 컴퓨터가 켜져 있고 Claude Desktop 앱이 실행 중이어야 작동합니다. 절전·앱 닫힘 상태에서는 폰에서 보낸 지시가 처리되지 않습니다. 파일 데이터 추출·메일 요약·문서 작성 등은 안정적이고, 일부 서드파티 앱 조작은 아직 불안정합니다.

💡 공식 정의 (Anthropic Cowork): "Message from your phone. Claude works on your computer. You get the result."
폰으로 보내고 · 컴퓨터가 일하고 · 결과를 받는 패턴 — 임장·외근이 많은 공인중개사 업무에 가장 잘 맞는 사용법입니다.
